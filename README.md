## ChatBI — 对话式数据查询与可视化

基于自然语言交互的商业智能分析平台，用户通过对话即可完成数据库查询与可视化分析。

| 项目 | 说明 |
|------|------|
| 后端 | Django + LangGraph (8 节点状态机) |
| AI 引擎 | Qwen2.5-32b-instruct (Ollama 本地 / DashScope 云端双模式) |
| 工具协议 | MCP (Model Context Protocol) — 同步 + 异步双客户端 |
| 前端 | Vue 2 + ElementUI + ECharts + DOMPurify |
| 安全 | SQL 校验拦截、XSS 防护、输入敏感词检测 |

**核心功能：**

- 自然语言驱动的数据库查询与 ECharts 图表自动生成
- 6 类错误智能分类 + 针对性引导建议
- 中英文双语界面，Dark/Light 双主题
- 工具调用透明化卡片，SQL 语法高亮
- 多轮对话 + 追问建议系统

## 星辰岛屿 (Xingchen Island)

基于 Unity 2022.3 LTS 开发的 3D 开放世界 RPG 游戏，玩家在奇幻岛屿上探索、战斗、收集资源、完成任务。

| 项目 | 说明 |
|------|------|
| 引擎 | Unity 2022.3 LTS + URP 渲染管线 |
| 语言 | C# (.NET Standard 2.1) |
| 架构 | ServiceLocator + Bootstrap 四阶段启动 + InputArbiter 输入仲裁 |
| 视角 | 第一/第三人称自由切换 |

**核心系统：**

- **背包系统** — MVC 分层：Model / Service / View，支持拖拽、装备、消耗品、负重管理
- **战斗系统** — EnemyBase 泛型基类 + NavMeshAgent + 多敌人类型 AI 状态机
- **任务系统** — 事件驱动 QuestManager + ScriptableObject 数据定义（收集/击杀/交互/探索/交付）
- **交互系统** — 统一 IInteractable 接口，树木采集、果实采摘、虫类捕捉、Qi 碎片收集
- **昼夜循环** — 120 秒日夜交替，影响敌人刷新与场景氛围
- **导航系统** — 鼠标右键 ClickToMove + NavMesh 寻路

## 基于 YOLO 的计算机视觉目标检测系统

基于 YOLOv11 的实时目标检测系统，用户上传图片即可自动识别并标注物体。

| 项目 | 说明 |
|------|------|
| 模型 | YOLOv11 (Ultralytics) |
| 后端 | Flask (Python) |
| 前端 | Uni-app (Vue 3 跨平台框架) |
| 部署 | requirements.txt / environment.yml 一键部署 |

**核心功能：**

- 图片上传与实时目标检测
- 检测结果可视化为标注图像（边界框与类别标签）
- 跨平台前端支持 H5 与移动端
