## ChatBI 聊天式商业智能系统

基于自然语言交互的商业智能分析平台，用户通过对话即可完成数据查询与可视化分析。

### 技术栈

- **后端**: Django 6.0 (Python)
- **前端**: Vue 2 + ElementUI + ECharts
- **协议集成**: MCP (Model Context Protocol)
- **安全**: DOMPurify 防御 XSS 攻击
- **数据**: SQLite

### 核心功能

- 自然语言驱动的数据问答
- 支持柱状图、饼图、折线图等多种 ECharts 图表自动生成
- Markdown 格式对话渲染
- MCP 协议实现工具调用与智能体协作

### 架构亮点

- 后端采用分层设计：controller 层处理请求，agents 层承载智能体逻辑，tools 层封装工具函数，mymcp 模块实现 MCP 协议集成
- 前端基于纯 HTML 构建单页应用，通过 CDN 引入 Vue 2、ElementUI、ECharts、Axios 等依赖，无需构建工具链即可运行
- 前后端通过 RESTful API 通信，Axios 处理异步请求，适配多种业务场景

## 基于 YOLO 的计算机视觉目标检测系统

基于 YOLOv11 的实时目标检测系统，用户上传图片即可自动识别并标注物体。

### 技术栈

- **模型**: YOLOv11 (Ultralytics)
- **后端**: Flask (Python)
- **前端**: Uni-app (Vue 3 跨平台框架)
- **部署**: 支持 requirements.txt 和 environment.yml 一键部署

### 核心功能

- 图片上传与实时目标检测
- 检测结果可视化为标注图像（边界框与类别标签）
- 跨平台前端支持 H5 与移动端

### 架构亮点

- 后端基于 Flask 提供 `/ai_detect` API，加载 `yolo11n.pt` 预训练模型，接收图片后执行推理并返回检测结果
- 前端使用 Uni-app 框架实现一套代码多端运行，深色炫酷 UI 风格
- 提供完整的 Python 环境配置与部署脚本，降低运行门槛
